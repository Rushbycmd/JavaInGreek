# Example code in Greek!:


```java
δημόσια τάξη Example {
    δημόσια στατική κενό main(String[] args) {
        χόνδρος αλφάβητος = "abcdefghijklmnopqrstuvwxyz";
        αν (αλφάβητος.ισοδυναμεί("alfabet")) {
            System.out.αποτύπωσε("αληθής");
        } αλλιώς {
            System.out.αποτύπωσε("ψευδής");
        }
    }
}
```

# All supported polish words:
`καθώς` - `while`

`αν` - `if`

`αληθής` - `true`

`ψευδής` - `false`

`δημόσια` - `public`

`ιδιωτικό` - `private`

`προστατευμένο` - `protected`

`στατική` - `static`

`κενό` - `void`

`διακοπή` - `break`

`συνέχισε` - `continue`

`άλλαξε` - `switch`

`τάξη` - `class`

`χόνδρος` - `String`

`τελικός` - `final`

`μηδενικό` - `null`

`δοκίμασε` - `try`

`άρπαξε` - `catch`

`ισοδυναμεί` - `equals`

`αποτύπωσε` - `print`

`αλλιώς` - `else`
