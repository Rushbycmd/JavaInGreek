# Example code in polish:

```java
publiczna klasa Example {
    publiczna statyczna próżnia main(String[] args) {
        sieć alfabet = "abcdefghijklmnopqrstuvwxyz";
        jeżeli (alfabet.równaSię("alfabet")) {
            System.out.drukujln("prawda");
        } w przeciwnym razie {
            System.out.drukujln("falsz");
        }
    }
}
```

# All supported polish words:
`gdy` - `while`

`jeżeli` - `if`

`prawda` - `true`

`fałsz` - `false`

`publiczna` - `public`

`prywatna` - `private`

`chroniona` - `protected`

`statyczna` - `static`

`próżnia` - `void`

`zniszcz` - `break`

`kontynuuj` - `continue`

`przełącz` - `switch`

`klasa` - `class`

`sieć` - `String`

`finalna` - `final`

`zero` - `null`

`próba` - `try`

`złap` - `catch`

`równaSię` - `equals`

`drukuj` - `print`

`w przeciwnym razie` - `else`
